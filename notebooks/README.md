Componente de Investigación y Modelado 

Este directorio contiene el cuaderno interactivo (.ipynb) que documenta el proceso científico de descubrimiento de datos (Knowledge Discovery in Databases - KDD).

 Contenido del Notebook

El script Proyecto_parte_1.ipynb detalla el paso a paso metodológico:

Ingesta y ETL: Protocolos de limpieza de datos y conversión de tipos.

Análisis Exploratorio: Histogramas logarítmicos y análisis de frecuencias.

Ingeniería de Características: Transformación de datos transaccionales a vectores por entidad.

Entrenamiento:

Estandarización Z-Score.

Selección de K mediante el Método del Codo.

Entrenamiento de K-Means.

Validación: Cálculo del Coeficiente de Silueta (Resultado obtenido: 0.871).

 Ejecución

Este archivo está optimizado para ejecutarse en Google Colab o Jupyter Lab.
No requiere instalación local si se visualiza en GitHub, pero para ejecutarlo localmente se requieren las librerías listadas en requirements.txt.
